Reflection 3 ( Heroku and Github )

As Miss Jen said that we will going to publish our created website in Heroku, I got excited cause this can help me improve my Web Design technique with the help of some users, who can criticize my design and give any remarks or suggestion to better present my project.

It is very hard when uploading to GitHub to Heroku. I seek the help of my classmates to help me upload my html to GitHub and publish to Heroku. I’m not satisfied in my site upon seeing my classmates’ creations. I updated my site but not published yet.
At the end, I learned to appreciate GitHub and Heroku after sometimes uploading my site.
