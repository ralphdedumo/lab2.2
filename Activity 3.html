Activity 3 ( Design a 1000 floor Elevator )

I design an elevator interface for a 1000 floor by an AI interface with face and voice recognition. I know this kind of technology would have a many problem.

The elevator has also the old fashion number, but not in buttons. It has a scrollable number of floors, associated with voice face recognition. When the elevator is moving, the scrollable number of floors will switch to the face recognition that will display all the faces in that elevator.
The voice recognition of the elevator will work anytime by the power of the AI, the user can interact with the AI to stop (but there is actual emergency button), play music, etc
I want to implement this with the help of Burj Khalifa’s current elevator technology because it only takes 8 minutes to go from bottom to top, so with this improvement to that elevator, it becomes easier to the users of the elevator.
